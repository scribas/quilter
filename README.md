## Dependencies

Please make sure you have these dependencies first before building.

```bash
valac
libgranite-dev
libgtkspell3-3-dev
gtk+-3.0
gtksourceview-3.0
libwebkit2gtk-4.0-dev
libmarkdown2-dev
libgtkspell3-3-dev
meson
```

## Building

Simply clone this repo, then:

```bash
meson build && cd build
meson configure -Dprefix=/usr
sudo ninja install
```
